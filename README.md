Studi Kasus 1:

Pada studi kasus ini, saya menggunakan database yang memiliki 2 tabel, yaitu tabel user dan tabel penjualan.

***Tabel user***

| Name | Type |
| ------------- | ------------- |
| ID_User  | int  |
| Email | varchar(30) |
| Password | varchar(20) |

***Tabel penjualan***
| Name | Type |
| ------------- | ------------- |
| ID_Penjualan  | int  |
| Bulan | varchar(40) |
| Tahun | int |
| Hasil_Penjualan | int |
